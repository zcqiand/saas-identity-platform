# saas-identity-platform — 功能规格文档

> 本文档描述 saas-identity-platform 的所有功能特性、验收标准与测试策略。
> 与 [CLAUDE.md](CLAUDE.md) 共同构成本仓的完整规范文档。

## 功能清单

### 多租户架构

| 功能名称 | 功能描述 | 验收标准 |
| :--- | :--- | :--- |
| 租户上下文 TenantContext | React Context 提供当前租户信息 | 子组件通过 useTenant() 获取当前租户 ID |
| 租户 store | tenantStore 管理 init/switch/subscribedFeatures | 切换租户后所有 API 请求携带正确的 X-Tenant-ID |
| 租户布局 TenantLayout | 租户级别 Layout，含侧边栏/顶栏 | 平台租户与普通租户布局分离 |
| 租户切换器 TenantSwitcher | 下拉切换活跃租户 | 切换后刷新路由和状态 |
| 租户主题 | 各租户独立主题色配置 | theme.ts 提供 applyTheme/clearTheme |

### 统一认证（SSO/OAuth2）

| 功能名称 | 功能描述 | 验收标准 |
| :--- | :--- | :--- |
| SSO 授权跳转 | 构建 SSO 授权 URL，重定向至 IdP | MSW mock /sso/authorize 端点返回 302 回调 |
| OAuth2 回调 | /sso/oauth/callback 接收 code，换 token | MSW 签发 mock access_token |
| 多种 OAuth Provider | 支持 5 种 OAuth provider | buildOAuthAuthorizeUrl 正确拼接各 provider 参数 |
| JWT 签发/校验 | mock JWT 签发（固定密钥，非生产凭证） | loginWithSso 返回 JWT 并写入 authStore |

### RBAC 权限管理

| 功能名称 | 功能描述 | 验收标准 |
| :--- | :--- | :--- |
| 角色 CRUD | RoleList / RoleFormModal 创建/编辑/删除角色 | 角色含 name/description/permissions |
| 权限矩阵 MenuPermissions | 角色×菜单权限矩阵配置 | 保存后 GET /auth/permissions 返回更新后的权限集 |
| 权限组 PermissionGroup | 权限分组管理 | PermissionGroupList 正确渲染 |
| 岗位 Position | 岗位管理（关联角色） | PositionList 含岗位 CRUD |
| 用户组 UserGroup | 用户分组管理 | UserGroupList 含用户组 CRUD |

### 用户与组织管理

| 功能名称 | 功能描述 | 验收标准 |
| :--- | :--- | :--- |
| 用户 CRUD | UserList / UserFormModal 创建/编辑/删除用户 | 用户关联租户/角色/组织 |
| 组织树 OrgTree | 递归组织树组件，支持展开/折叠/选中 | OrgNodeFormModal 编辑组织节点 |
| 审计日志 AuditLog | 审计日志分页 + action 类型过滤 | GET /audit-logs 返回分页数据 |

### 虚拟滚动

| 功能名称 | 功能描述 | 验收标准 |
| :--- | :--- | :--- |
| VirtualList 组件 | react-window 定高虚拟窗口 | AuditLog 列表超过 100 项时仅渲染可视区域 |
| 虚拟滚动集成 | AuditLog 使用虚拟滚动 + shallowRef | 滚动流畅，无白屏/闪烁 |

### API Keys 与应用

| 功能名称 | 功能描述 | 验收标准 |
| :--- | :--- | :--- |
| API Key 管理 | ApiKeyList 创建/查看/撤销 API Key | Key 与租户/用户关联 |
| 应用管理 AppList | 应用 CRUD，含菜单配置 | AppList 显示所有已注册应用 |

### 监控与可观测性

| 功能名称 | 功能描述 | 验收标准 |
| :--- | :--- | :--- |
| Sentry 错误监控 | @sentry/react 集成，DSN 留空时 no-op | initSentry() 不报 console.error；captureError 正常上报 |
| Web Vitals | LCP/CLS/INP/FCP/TTFB 采集，上报 /api/vitals | MSW 拦截 /api/vitals 请求 |
| Lighthouse 目标 | Performance/Accessibility/Best Practices/SEO 均 > 90 | lighthouse http://localhost:3000 评分达标 |

### 平台级配置

| 功能名称 | 功能描述 | 验收标准 |
| :--- | :--- | :--- |
| 登录方式 LoginMethods | 配置登录方式（账号密码/SSO/OAuth） | 路由 /login-methods 可访问 |
| 登录安全 LoginSecurity | 密码策略/登录限制配置 | 路由 /login-security 可访问 |
| 通知配置 NotificationConfig | 通知渠道/模板配置 | 路由 /notification-config 可访问 |
| 开放平台 OpenPlatform | 第三方应用接入配置 | 路由 /open-platform 可访问 |
| 密码策略 PasswordPolicy | 密码复杂度/有效期配置 | 路由 /password-policy 可访问 |
| 平台配置 PlatformConfig | 平台全局配置 | 路由 /platform-config 可访问 |
| 平台租户管理 PlatformTenants | 平台级租户列表 | 路由 /platform-tenants 可访问 |
| 风控配置 RiskControl | 登录风控/异常检测规则 | 路由 /risk-control 可访问 |
| Token 配置 TokenConfig | Access Token/Refresh Token 策略 | 路由 /token-config 可访问 |

### 测试策略

| 层级 | 目录 | 说明 |
| :--- | :--- | :--- |
| 单元测试 | `tests/types/` `tests/msw/` `tests/components/` `tests/features/*/` | 类型契约、MSW handler、组件、store 隔离测试 |
| 集成测试 | `tests/integration/` | 多租户切换 + SSO 登录 + 权限守卫 + 用户管理全链路 |
| E2E 冒烟测试 | `src/test/` | 登录→业务页面→权限验证→租户切换 端到端 |
| 监控测试 | `tests/monitoring/` | Sentry DSN 空 no-op + web-vitals 采集验证 |
